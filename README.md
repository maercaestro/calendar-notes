# Cyberpunk Calendar Notes

A futuristic, interactive calendar application with a cyberpunk aesthetic. Add notes to specific dates and store them locally in your browser.

## Features

- Cyberpunk-themed UI with neon effects
- Interactive calendar grid for the entire year
- Add, edit, and delete notes for any date
- Local storage for data persistence
- Responsive design

## Deployment

This is a static website that can be deployed on Vercel.

### Manual Vercel Deployment

1. Push this repository to GitHub.
2. Go to [Vercel](https://vercel.com) and sign in.
3. Click "New Project" and import your GitHub repository.
4. Vercel will automatically detect it as a static site and deploy it.
5. Your calendar will be live at the provided Vercel URL.

## Usage

- Click on any date in the calendar to add or edit a note.
- Notes are saved automatically to your browser's local storage.
- Use the year navigation buttons to switch years.
- Click "Purge Memory" to clear all notes.
